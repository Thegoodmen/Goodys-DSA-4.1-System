# Goodys-DSA-4.1-System

## Current Features
- Automated Combat with NPC Actors
- Importing Characters from Heldentool
- Loot Actors, Merchant  Sheets and Shoping System
- Support for dice-so-nice and pdf Integration

## Upcoming Features
- Support for Activ Effects for a number of Spells
- GM-Tools and Roll Tables
- Summening Integration (Demonic&Elemental)
- Weapon & Artefact  Smiting

## Links & Recommendations
- [Dice so Nice](https://gitlab.com/riccisi/foundryvtt-dice-so-nice) is an amazing addon with wonderfull 3D Dices and fully supported by the System
- [PDF Pager](https://github.com/farling42/fvtt-pdf-pager) is a helpfull Tool to have you PDFs inside Foundry ready
